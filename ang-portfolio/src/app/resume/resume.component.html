Automated Data Cleaning (part of masters thesis)

Description:
Perhaps the most difficult part of this project was writing
the logic to detect irratic data and clean it. I wrote a script for the
Stata statistics package that would automatically go through all of the
responses by each participant and flag the data if there were
inconsistencies in responding. This algorithm would flag
inconsistencies, count the number of inconsistencies, and then
automatically recode data in which the inconsistency was deemed to be
trivial.

Future Direction:
I am currently working on a project that extends the work I did here.
See the Accessible Online Purchase Tasks project above for more details.

Punching Machine Project

Description:
This project used MATLAB, a FTDI UM245R USB-to-Parallel Interface, a
laser, and a photodiode, along with sections of PVC and wires that I put
together. The goal was to measure the speeed of fast punches and to
display the punch count and speed on the screen. The logic was based on
the signal from the photodiode, which was altered by breaking the laser
beam.
